debugger keyword


  (() => {
    const date = new Date();
    if(date.getFullYear() >= 1992) {
        if(date.getMonth() >= 9) {
            if(date.getDate() <= 13) {
              console.log(date.toISOString());
            }
        } else {
            debugger;
        }
    }
  })();

Press F12; The 'debugger' breakpoint pauses the execution when evaluated; While paused, try setting a breakpoint by clicking on it's line number.
